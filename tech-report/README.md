# Tech Report (Overleaf Files)

This folder stores an Overleaf-ready ACL report draft.

## What is inside

- `main.tex` - main ACL paper file
- `custom.bib` - bibliography entries
- `sections/` - section text files
- `figures/` - put PNG/PDF figures here

## How to use with Overleaf

1. Create a new Overleaf project from the ACL template.
2. Upload all files from this `tech-report/` folder.
3. Keep ACL style files from the Overleaf template.
4. Recompile.

## Course constraints to remember

- Use ACL long paper template.
- Final output name: `Last Name, First Name.pdf`
- Include: main results, ablation, and error analysis.
